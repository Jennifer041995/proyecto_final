Proyecto Universitario

Esta aplicación web es un proyecto desarrollado como parte del curso de Programación Web II de la Universidad Cristiana de las Asambleas de Dios (UCAD).

Descripción del Proyecto

Music UCAD es una aplicación web desarrollada con Angular y Firebase que permite gestionar una colección de música. El proyecto incluye funcionalidades como:

  • Gestión de canciones (CRUD completo)
  • Reproductor multimedia con Plyr
  • Autenticación de usuarios con Firebase
  • Almacenamiento de datos en tiempo real

Tecnologías Utilizadas

  • Angular 20
  • TypeScript
  • Firebase
  • Bootstrap 5
  • Plyr (Reproductor multimedia)
  • SweetAlert2
  • Font Awesome
